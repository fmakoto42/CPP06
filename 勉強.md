#### 入力文字列の判定

+-, ., f, 数字, アルファベット の数を計数
+- : 0 or 1
. : 0 or 1
f : 0 or 1

int : +- 0 or 1, . 0, f 0, アルファベット 0


(1) 一文字目が符号 [ +, - ] なら、符号フラグセットし先頭文字列を削除
(2) inf、nan であれば、種別フラグをセット
　その次が f ならfloat、終端ならdouble、そうでなければエラー
(3) . なら以降小数点以下
(4) 数字なら読み込み
